# Commit 6 — Jira Ticket Draft Generator

Adds a new report-like artifact: Jira ticket drafts.

Unlike Project Report, this feature generates actionable Jira issues from Project Memory built from:
- meeting videos
- Slack messages
- Jira PDFs
- Jira pasted text

The UI supports:
- number of tickets
- ticket style
- optional project context
- markdown download
- JSON download

Apply instructions are in `APP_CHANGE.md`.
