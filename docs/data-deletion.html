Удаление пользовательских данных

Дата последнего обновления: 21 июля 2026 года

Пользователь Project Brain может запросить удаление своей учётной записи и связанных с ней данных.

Как отправить запрос

  1. Отправьте письмо на yurgen01@gmail.com.
  2. Укажите тему письма: Удаление данных Project Brain.
  3. Укажите email, который использовался для входа в приложение.
  4. Если вход выполнялся через Facebook, сообщите об этом в письме. Не отправляйте пароль, токен или другие секреты.

Для защиты пользователя оператор может запросить подтверждение владения указанным email. Запрос будет обработан в течение 30 календарных дней после необходимого подтверждения.

Какие данные будут удалены

  • профиль и привязки к провайдерам входа;
  • проекты и пользовательские настройки;
  • загруженные источники, транскрипты и результаты обработки;
  • созданные пользователем артефакты.

Отдельные технические записи могут сохраняться в ограниченном виде, если это необходимо для безопасности, предотвращения злоупотреблений или выполнения применимых требований.

Отзыв доступа Facebook

Вы также можете удалить связь Project Brain с Facebook в настройках Facebook: «Настройки и конфиденциальность» → «Настройки» → «Приложения и сайты» → Project Brain → «Удалить». Отзыв доступа в Facebook не всегда удаляет уже сохранённые в Project Brain данные, поэтому для полного удаления отправьте запрос по инструкции выше.
